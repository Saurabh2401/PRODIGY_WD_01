# Interactive Navigation Menu

A modern and responsive navigation menu built using **HTML, CSS, and JavaScript**. The project features a fixed navigation bar, dynamic styling on scroll, smooth hover effects, active link highlighting, and a mobile-friendly responsive design.

## 🚀 Features

* Fixed navigation bar
* Dynamic color change on scroll
* Active menu highlighting
* Smooth hover animations
* Responsive mobile navigation
* Modern warm-themed UI
* Glassmorphism-inspired design
* Smooth scrolling experience

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd interactive-navigation-menu
```

3. Open `index.html` in your browser

No additional dependencies are required.
